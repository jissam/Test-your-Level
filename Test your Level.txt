Have you heard of HTML? If yes, what does HTML stand for, and what is its primary purpose in web development?
HTML est un langage utilisé pour structurer le contenu des pages Web. 
HTML permet aux navigateurs Web de l'interpréter et de l'afficher correctement pour les utilisateurs.

Are you familiar with CSS? If yes, briefly explain what CSS is used for in web development.
CSS est un langage utilisé dans le développement Web pour définir la présentation visuelle et le style des éléments HTML d'une page.

Have you heard of JavaScript? If yes, what is JavaScript, and why is it crucial for building interactive websites?

JavaScript est un langage de programmation utilisé dans le développement Web pour rendre les sites Web interactifs et dynamiques.
